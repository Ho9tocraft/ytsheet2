### 設置の手順 #######################################################################################
 1. config.cgi.defaultのファイル名をconfig.cgiに変更する（.defaultを消す）

 2. config.cgiを開いて、管理用パスなどを設定する

 3. ゆとシートのファイルをすべてサーバーにアップする

 4. index.cgiのパーミッション（属性）を<705>または<755>に変更する
   （サーバーによっては<700>などの可能性もアリ）

 5. ブラウザからシートのTOPページ(http://○○○/ytsheet_sw2.5/)へアクセスして
    問題なく表示されれば設置完了

### ファイル構成 #####################################################################################
  <>内は一般的なパーミッション設定です。基本的に、「index.cgi」以外は設定する必要はありません。

 /ytsheet_sw2.5 <705> ディレクトリ（フォルダ）名は変更可
  |
  |- index.cgi          <705> 基本CGI。パーミッション（属性）設定必須。
  |- config.cgi.default <604> 設定ファイル。ファイル名を「必ず」 config.cgi に変更してください
  |
  |- /skin <705> テンプレート格納ディレクトリ
  |  |
  |  |- chara.css  <604> キャラシートのCSS(基本)
  |  |- edit.css   <604> 編集画面用CSS
  |
  |- /lib <705> ライブラリ格納ディレクトリ
  |  |
  |  |- config-default.pl <604> デフォルト設定格納ファイル
  |  |- data-*****.pl     <604> 種族・特技データ等格納ファイル
  |  |- *****.pl          <604> その他処理ライブラリ
  |
  |- /data   <705> キャラクターデータ格納ディレクトリ
  |- /tmp    <705> その他データ格納ディレクトリ
